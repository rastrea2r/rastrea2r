.. _install-guide-label:

Install Guide
=============

.. note::

    It is best practice to install run Python projects in a virtual
    environment, which can be created and activated as follows using
    Python 3.6+.

    .. code-block:: console

        $ python -m venv venv
        $ source venv/bin/activate
        (venv) $ pip install rastrea2r

The simplest way to install rastrea2r is using Pip.

.. code-block:: console

    $ pip install rastrea2r

This will install ``rastrea2r`` and all of its dependencies.

Report Bugs
===========

Report bugs at the `issue tracker <https://github.com/ssbhat/rastrea2r/issues>`_.

Please include:

  - Operating system name and version.
  - Any details about your local setup that might be helpful in troubleshooting.
  - Detailed steps to reproduce the bug.
